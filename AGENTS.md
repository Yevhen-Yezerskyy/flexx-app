# Agent Rules

- Любые действия с кодом только после явной команды пользователя.
- Любые действия с кодом только после прямого вопроса к пользователю и его подтверждения.
- Ответы максимально краткие: по возможности одно предложение; два хуже; три еще хуже.
- Никогда не гадать: сначала читать код проекта и опираться только на проверенный контекст.
